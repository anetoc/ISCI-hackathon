# ISCI — Immune-State Controllability Index (summary)

Genome-scale Perturb-seq shows which genes *change* T-cell state, but not which genes *control* it. We asked whether controllership is separable from raw effect size in the Marson CD4+ T-cell screen. Testing an index *against* effect magnitude failed: known regulators simply have ~99× larger effects, so magnitude wins by construction. Reframing the question — does a feature add signal *conditional on* magnitude? — revealed a genuine, magnitude-independent signal: regulators produce effects that are more **state-specific** (concentrated on a functional axis) and more **reproducible across donors**. This orthogonal signal nearly doubles regulator recovery (AUPRC 0.41→0.64; bootstrap gain +0.23, P>0=99.6%), survives leakage controls, and replicates across three culture conditions. As a sanity check it recovers known regulators near the top (IRF1, STAT6, GATA3, SETDB1); as discovery it nominates novel candidates not in the label set — IKBKB (NF-κB), ZC3H12A (Regnase-1), RCOR1, BCLAF1, PRKDC — each with a cited evidence card. The broader lesson — in-dataset controllability benchmarks are magnitude-confounded and require a conditional or external test — is a reusable caution for the perturbation-genomics field.
